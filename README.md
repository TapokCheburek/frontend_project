## Установка и локальный запуск

Все команды должны выполняться из папки **`app_chat`**.

Откройте терминал и перейдите в папку `app_chat`:

```bash
cd app_chat
```

### 2. Установка зависимостей

Убедитесь, что у вас установлены все необходимые npm-пакеты перед первым запуском:

```bash
npm install
```

### 3. Запуск проекта (режим разработчика)

Для запуска локального сервера разработки используйте команду:

```bash
npm run dev
```

Приложение будет доступно в браузере (http://localhost:5173).

### 4. Запуск тестов

В проекте настроено окружение **Vitest** + **React Testing Library**. Чтобы прогнать все написанные unit- и компонентные тесты, используйте:

```bash
npm run test
```

Эта команда проверит редьюсеры, UI-компоненты панелей, чата, а также логику хранилища (`localStorage`).

---

**Примечание:** Для генерации ответов от нейросети в приложении может потребоваться API ключ (Gemini). Вы сможете указать его через интерфейс самого чата.
